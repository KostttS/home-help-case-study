# Home Help — Home Services Mobile App

Mobile app for finding home service specialists and requesting help with household tasks.

The app was built around a simple client flow: choose a service, find a master, contact directly and solve the problem without unnecessary complexity.

## Technical challenges

### Simple service request flow

The main goal was to make the app practical for users who need quick help at home.

Instead of a complicated marketplace flow, the app focused on fast service selection, direct contact and a clear path from problem to specialist.

### Client and master roles

The app required separate logic for clients and masters, with different data, screens and access rules.

### Production support

The project required real production support, including backend checks, app availability, server/database monitoring and fixing issues affecting real users.
